Описание главного меню

WI-FI

WI-FI: Переключатель для включения/выключения WI-FI.

Отключить WI-FI через: Промежуток времени на который включается WI-FI при включении устройства или перезагрузке.

Номер канала: Выбор номера канала WI-FI.

SSID: Ввод имени точки доступа WI-FI.

Пароль: Установка пароля точки доступа WI-FI.

После завершения всех изменений необходимо нажать кнопку "Сохранить" в правом нижнем углу. Для возврата в основное меню необходимо нажать кнопку "Назад" в левом нижнем углу.

ЗАПИСЬ

Установка времени: Ручная установка времени и даты, настройка режимов день/ночь.

Изменить время: Установка времени и даты в случае неактуального времени на устройстве. Неправильные время и дата возможны при отсутствии интернета.

Время дня: Настройка режимов день/ночь (голубой промежуток-день, чёрный-ночь). Этот параметр может применяться для оптимизации трафика интернета. Для изменения значений необходимо передвинуть ползунки влево или вправо, определяя желаемые границы дня и ночи.

Запись в облачное хранилище: Параметры устройства для записи в облачное хранилище.

Запись в облачное хранилище: Включить/выключить запись в облачное хранилище.

Настройки облачного сервиса: Параметры облачного сервиса на который будет производиться запись.

Сервис: Выбор облачного сервиса для записи.

Логин: Логин учётной записи на выбранном сервисе.

Пароль: Пароль учётной записи на выбранном сервисе.

Настройки записи по движению: Группа параметров записи по движению.

День: Параметры записи по движению днём. Параметры активны в голубом промежутке времени установленном в пункте "Время дня".

Запись по движению: Переключатель включения/выключения записи по движению днём.

Длина записи: Длина видеоролика, записываемого по движению днём. Минимальное значение - 1 секунда, максимальное - 10 секунд.

Ночь:Параметры записи по движению ночью. Параметры активны в чёрном промежутке времени установленном в пункте "Время дня".

Запись по движению: Переключатель включения/выключения записи по движению ночью.

Длина записи: Длина видеоролика, записываемого по движению ночью. Минимальное значение - 1 секунда, максимальное - 10 секунд.

Настройки периодической записи: Группа параметров записи по времени.

День: Параметры записи по времени днём. Параметры активны в голубом промежутке времени установленном в пункте "Время дня".

Периодическая запись: Переключатель включения/выключения записи по времени днём.

Период записи: Интервал периодической записи видеороликов днём в секундах. Минимальное значение 60 секунд.

Длина записи: Длина видеоролика, записываемого днём по времени. Минимальное значение 1 секунда, максимальное - 10 секунд.

Ночь:Параметры записи по времени ночью. Параметры активны в чёрном промежутке времени установленном в пункте "Время дня".

Периодическая запись: Переключатель включения/выключения записи по времени ночью.

Период записи: Интервал периодической записи видеороликов ночью в секундах. Минимальное значение 60 секунд.

Длина записи: Длина видеоролика, записываемого ночью по времени. Минимальное значение 1 секунда, максимальное - 10 секунд.

После завершения всех изменений необходимо нажать кнопку "Сохранить" в правом нижнем углу. Для возврата в основное меню необходимо нажать кнопку "Назад" в левом нижнем углу.

ПОМОЩЬ

Помощь пользователю по навигации с описанием отдельных функций.

Для возврата в основное меню необходимо нажать кнопку "Назад" в левом нижнем углу.

КАМЕРА

Камера: Поле ввода строки адреса RTSP потока подключенной IP камеры.

Чувствительность: Параметр определяющий паузу в секундах при записи видеороликов. Применяется в режиме записи по движению. Позволяет экономить интернет трафик при постоянном или частом движении в кадре.

После завершения всех изменений необходимо нажать кнопку "Сохранить" в правом нижнем углу. Для возврата в основное меню необходимо нажать кнопку "Назад" в левом нижнем углу.

3G/4G

Информационная страница позволяет оценить уровень сигнала мобильной сети.

Шкала отображает уровень сигнала в процентах и в цветах: зелёный - хороший уровень, жёлтый - удовлетварительный уровень, красный - плохой уровень.

Далее отображается дополнительная информация по сети и оборудованию.

Значения обновляются каждые 3 секунды, также можно обновить вручную с помощью кнопки "обновить" в правом нижнем углу.

При отсутствии 3g/4g модема в устройстве появится надпись "3G/4G-модема не обнаружено".

Для возврата в основное меню необходимо нажать кнопку "Назад" в левом нижнем углу.

ХРАНИЛИЩЕ

Переход на страницу облачного сервиса выбранного в меню "Запись->Запись в облачное хранилище->Настройки облачного сервиса->Сервис".

При первом посещении необходимо ввести логин и пароль облачного сервиса, которые должны совпадать с введёными в пункте "Запись->Запись в облачное хранилище->Настройки облачного сервиса->Сервис" значениями.
